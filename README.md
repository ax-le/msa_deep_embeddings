Drop of the code for the SMC submission "Unsupervised Evaluation of Deep Audio Embeddings for Music Structure Analysis". The code is not properly formatted for open-source use, and may be unstable, sorry about that. In near future, I will make it better, but, as of now, experiments may be reproduced.

Code for segmentation is contained in the 'as_seg' package (https://gitlab.imt-atlantique.fr/a23marmo/autosimilarity_segmentation).
